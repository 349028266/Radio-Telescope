# Radio-Telescope
a project  to create a rudimentary Radio telescope. This project aims to create a functional radio telescope that can recieve a decent amount of data from the cosmos.

The dipole antenna is only stage one of the development. I will attempt to use different antennas (i.e. parabolic reflector) that will enhance the telescope later due to their difficulty in construction. For now, I will use only the dipole antenna.

This project uses a dipole antenna constructed out of wire, fed into a balun, then fed again into a SDR Reciever raspberry Pi to then be processed.
